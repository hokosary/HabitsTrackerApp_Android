# Habit Tracker App

Простое мобильное приложение для отслеживания привычек, разработанное с использованием React Native и Expo.

## Инструкция по запуску

### Клонирование репозитория
1. Откройте терминал.
2. Выполните команду для клонирования репозитория (замените URL на ваш):
   ```bash
   git clone https://github.com/your-username/HabitTracker.git
   ```
3. Перейдите в папку проекта:
   ```bash
   cd HabitTracker
   ```

### Установка зависимостей и запуск
1. Установите все необходимые зависимости проекта:
   ```bash
   npm install
   ```
2. Запустите приложение с помощью Expo:
   ```bash
   npx expo start
   ```
   Или:
   ```bash
   npm run start
   ```
3. После запуска команды появится QR-код. 
   - Для запуска на **Android**: установите приложение Expo Go из Google Play, откройте его и отсканируйте QR-код.
   - Вы также можете нажать `a` в терминале, чтобы запустить приложение в эмуляторе Android (если он у вас установлен и настроен).

### Сборка APK файла
Для получения готового `.apk` файла, который можно установить на любое Android-устройство, используйте облачную сборку Expo Application Services (EAS):

1. Установите EAS CLI (если еще не установлен):
   ```bash
   npm install -g eas-cli
   ```
2. Авторизуйтесь в своем аккаунте Expo (потребуется создать аккаунт на expo.dev, если его нет):
   ```bash
   eas login
   ```
3. Запустите процесс сборки Android APK:
   ```bash
   eas build -p android --profile production
   ```
4. После завершения сборки терминал выдаст ссылку на скачивание вашего `.apk` файла. Вы также можете найти его в панели управления на сайте [expo.dev](https://expo.dev).
